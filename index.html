About Dashboard Manager Add-on

Dashboard Manager is a lightweight Google Sheets add-on built exclusively for the Pro Bettors Academy community. It automates monthly sheet creation, log tracking, and cleanup—so you can focus on strategy, not spreadsheet maintenance.

Key Features

  • Create Month: Instantly generate a new monthly sheet + log with one click.
  • Show Month: Hide all but the current month for a clean, focused workspace.
  • Delete Month: Remove old months safely while preserving your master dashboard.
  • Custom Menu: Access everything via the “Dashboard Manager” menu in Google Sheets.

Your Privacy Comes First

This add-on only accesses the exact copy of the Pro Bettors Academy Dashboard you have open. It cannot see, touch, or access any other file in your Google Drive — guaranteed by Google’s strictest limited scope.

Google-Reviewed Permissions (Narrowest Possible)

https://www.googleapis.com/auth/spreadsheets.currentonly

• View and manage only the spreadsheet this add-on is installed in (the one you opened)

https://www.googleapis.com/auth/script.container.ui

• Add menus & dialogs inside that sheet only

Zero Risk – 100 % Contained

Your data never leaves Google. The add-on is hard-limited by Google to the single spreadsheet you’re using — nothing else is possible, even if we wanted it to be. This is enforced by the official @OnlyCurrentDoc restriction and the .currentonly scope.
